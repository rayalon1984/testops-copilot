# API Specifications and Contracts

## Endpoint: GET /api/example
### Description
This endpoint retrieves example data.
### Request Parameters
- `param1`: The first parameter (required)
- `param2`: The second parameter (optional)
### Response
```json
{
  "data": [
    {
      "id": 1,
      "name": "example1"
    },
    {
      "id": 2,
      "name": "example2"
    }
  ]
}
```
