# System Architecture Documentation

## Overview
This document outlines the system architecture for the TestOps Companion application.

## Components
- **Frontend:** Built with React, providing a user-friendly interface.
- **Backend:** Node.js and Express for handling requests and serving the frontend.
- **Database:** MongoDB for storing application data and user information.

## Deployment
The application is deployed on AWS, utilizing EC2 instances for the backend and S3 for static file hosting.
