So I asked Claude to help me analyze a failing test.

I thought: "There's gotta be a better way."

Turns out there is. It's called Model Context Protocol (MCP).

Instead of sending your entire codebase to an LLM every single time, you give it focused *tools* that run server-side.

So I built an MCP server for test failure analysis.

The results?

It's like giving your AI a memory. And a calculator. And direct access to your database.

Pretty cool, right?

Built with TypeScript, PostgreSQL, and a lot of coffee ☕
