# Plans

This directory contains dated planning documents for TestOps Companion.

## Naming Convention

```
YYYYMMDD-short-description.md
```

## Status Tags

Each plan has a status in its frontmatter:
- **DRAFT** — Under development, not yet approved
- **APPROVED** — Approved for implementation
- **IN_PROGRESS** — Currently being implemented
- **COMPLETED** — Fully implemented and verified
- **ARCHIVED** — Superseded or no longer relevant
